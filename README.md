# Rock, Paper Scissors </br>
A simple CLI app that I made when I was first learning how to code.</br>
Try winning!</br>
## Features
* Playable via the command line
* Simple logic to determine the winner
* Immediate feedback
## Installation & Running
1. Download the python file
2. Navigate to the location it is in
3. Run the command: </br>
```python rps.py```
## Usage
Type either ```Rock``` , ```Paper``` or ```Scissors``` </br>
The computer will let you know if you won or lost
## What did I learn
* Introduction to Python
* How to use ```If```, ```else``` and ```elif``` statements
* How to import modules into Python
* Basic user input handling
